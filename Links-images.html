<p>
The paragraph tag defines a paragraph of text. It automatically adds some space (margin) before and after each paragraph element, making text content more readable and structured.
<a>
The anchor tag creates hyperlinks that connect one web page to another. It uses the href attribute to specify the destination URL, allowing users to navigate between pages or resources on the web.
<img>
The image tag embeds images into an HTML document. It uses the src attribute to specify the image source and the alt attribute to provide alternative text for accessibility and when images cannot be displayed.
